
# Bloggie

This is a ASP.NET MVC project for blogging.


## Abilities
* Registering anf login
* Multiple user levels (User, Admin, SuperAdmin)
* WYSIWYG editor
* Like and Comment for blogs
* User management
* Server-side and Client-side validation
